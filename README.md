# CSC425Project-Sentiment-Analysis
# Fall 2025 Artificial Intelligence Project: Sentiment Analysis - Chat GPT Reviews


<img width="1920" height="1080" alt="sentiment_analysisproj" src="https://github.com/user-attachments/assets/8345e6bd-f2c9-467e-855e-c143608de06c" />


Performed large-scale sentiment analysis with 60,000 ChatGPT user reviews using NLP 
preprocessing with trained ML models for classification. Engineered features to improve model 
and sentiment trends.

Trained ML models(Naive Bayes, Decision Tree, Gradient Boosting) for 
multi-class sentiment classification

Achieved 85% accuracy with evaluated performance using precision, 
confusion matrices, and cross-validation.

Kaggle Dataset: https://www.kaggle.com/datasets/ashishkumarak/chatgpt-reviews-daily-updated/data
(updated daily from Google Play Store ChatGPT Reviews)
